# Icone Estensione

Questa cartella deve contenere 3 icone PNG:

- `icon16.png` - 16x16 pixel
- `icon48.png` - 48x48 pixel  
- `icon128.png` - 128x128 pixel

## Opzioni per Creare le Icone

### 1. Generatore Online (Più Facile)
- Vai su https://www.favicon-generator.org/
- Carica un'immagine o usa un emoji 🤖
- Scarica le dimensioni richieste
- Rinomina i file e mettili qui

### 2. Usa Emoji come Icona
- Vai su https://favicon.io/emoji-favicons/robot/
- Scarica il pacchetto
- Usa le immagini nelle dimensioni 16x16, 48x48, 128x128

### 3. Crea con Canva/Figma
- Crea un design semplice con sfondo arancione (#ff6b00)
- Aggiungi testo "AI" o un'icona robot
- Esporta in 3 dimensioni

### 4. Icone Temporanee (Per Test)
Puoi usare qualsiasi immagine PNG nelle dimensioni richieste per testare l'estensione.

## Colori Suggeriti

- Primario: #ff6b00 (arancione)
- Secondario: #ffffff (bianco)
- Sfondo: #2d3436 (grigio scuro)
