# Readability.js

Questa cartella deve contenere il file `Readability.js` di Mozilla.

## Download

Scarica da uno di questi link:

1. **GitHub**: https://github.com/mozilla/readability/blob/main/Readability.js
   - Clicca su "Raw" e salva il file qui

2. **NPM/unpkg**: https://unpkg.com/@mozilla/readability@latest/Readability.js
   - Salva la pagina come `Readability.js` in questa cartella

3. **Via NPM** (se hai Node.js):
   ```bash
   npm install @mozilla/readability
   cp node_modules/@mozilla/readability/Readability.js lib/
   ```

## Verifica

Dopo il download, questa cartella dovrebbe contenere:
- `Readability.js` (circa 50KB)
- Questo file README.md

L'estensione non funzionerà senza questo file!
